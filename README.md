Movie Ticket Booking System
A lightweight, console-based Movie Ticket Booking System written in C++. This project simulates a real-world cinema booking experience, allowing users to view currently running movies, select showtimes, choose seats, and generate digital booking receipts.

Designed with clean Object-Oriented Programming (OOP) principles, this repository serves as an excellent demonstration of core C++ concepts, file handling, and data management.

🚀 Features
🎬 Movie Management: Browse a dynamic list of available movies, genres, and ratings.

💺 Interactive Seat Selection: View a visual representation of the seating layout (available vs. booked seats).

🎟️ Booking & Billing: Select multiple tickets, calculate total costs (including taxes), and generate a clean summary receipt.

💾 Data Persistence (Optional): Saves booking records and seat availability to .txt files so data isn't lost when the program closes.

🛠️ Admin Panel: Password-protected access to add/remove movies or reset seating charts.

🛠️ Tech Stack & Concepts Used
Language: C++11 (or higher)

Paradigm: Object-Oriented Programming (Classes, Objects, Encapsulation, and Inheritance)

File Handling: fstream library for saving and loading booking data.

Data Structures: Vectors/Arrays for seat matrices and movie lists.
