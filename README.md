Movie Ticket Booking System
<br>
A lightweight, console-based Movie Ticket Booking System written in C++. This project simulates a real-world cinema booking experience, allowing users to view currently running movies, select showtimes, choose seats, and generate digital booking receipts.
<br>
Designed with clean Object-Oriented Programming (OOP) principles, this repository serves as an excellent demonstration of core C++ concepts, file handling, and data management.
